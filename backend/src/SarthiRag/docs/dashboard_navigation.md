## **Platform Architecture & Navigation Overview**

Hirekruit is organized into a clean, intuitive dashboard. Saarthi must be able to guide users through the following sidebar sections:

### **Sidebar Modules**

* **Drive Creation:** The core starting point for launching new recruitment campaigns.  
* **Drives:** A centralized view of all active, paused, completed, and archived drives.  
* **All Applicants:** A global candidate database searchable by Job ID, name, or status.  
* **Shortlisted:** Repository for candidates who cleared initial automated and manual screening.  
* **Selected:** The final stage for candidates ready for offer letters and onboarding.  
* **Analytics:** Real-time dashboards showing funnel metrics, drop-off analysis, and time-to-hire.  
* **Calendar:** Integrated scheduling for assessment dates and interview slots.  
* **Notifications:** Real-time alerts for round completions and system updates.  
* **Settings:** Management of API keys, team permissions, and account logout.
